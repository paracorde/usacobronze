# Arrays

## Initialization

```python
foo = []
```

```java
int[] foo = new int[3];
```

```c++
int foo[3];
```

|0 |1 |2 |
|--|--|--|
|~ |~ |~ |

## Setting values

```c++
foo[0] = 3;
```

|0 |1 |2 |
|--|--|--|
|3 |~ |~ |

## Setting all values

```c++
for (int i = 0; i < foo.size(); i += 1){
   foo[i] = -1;
}
```

## Sorting

```python
foo = sorted(foo)
# or
foo.sort()
```

```java
Arrays.sort(foo)
```

```c++
sort(foo, foo+foo.size());
```

|0 |1 |2 |3 |
|--|--|--|--|
|3 |4 |1 |2 |

|0 |1 |2 |3 |
|--|--|--|--|
|1 |2 |3 |4 |
