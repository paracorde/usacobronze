# Loop Controls

## Continue

Java
```java
for (int i = 0; i < 10; i += 1){
   if (i == 4){
      continue;
   }
   System.out.println(i);
}
```

C++
```c++
for (int i = 0; i < 10; i += 1){
   if (i == 4){
      continue;
   }
   cout << i << "\n";
}
```

Python
```python
for i in range(10):
   if i == 4:
      continue
   print(i)
```

## Continue exercise

Write a program that checks if a given number is prime or not.

Then, write a program that prints out all the numbers 1..n that are not prime (take in n as input).
